Units Analysis

Author: DocNan Date: July 5, 2018  22:50:17

Units of normal physical values [1]

7 Basic units (SI units)

  • Mass: M [kg] (kilogram)

  • Time: T [s] (second)

  • Length: L [m] (metre)

  • Temperature: Θ\ThetaΘ [K] (Kelvin)

  • Amount of substance: N [mol] (mole)

  • Current: I [A] (Ampere)

  • Luminous intensity: J [cd] (candela)

Induced units

  • Frequency: f=1/Tf = 1/Tf=1/T -> [Hz] = [s^-1] (Hertz)

  • Force: F⃗=md2r⃗dt2\vec{F} = m\frac{d^2\vec{r}}{dt^2}​F​⃗​​=m​dt​2​​​​d​2​​​r​⃗​​​​ -> [N] = [kg .m .s^-2] (Newton)

  • Energy: W=F∗LW = F*LW=F∗L -> [J] = [N .m] = [kg .m^2 .s^-2] (Joule)

  • Pressure: P=F/SP = F/SP=F/S -> [Pa] = [N .m^-2] = [kg .m^-1 .s^-2] (Pascal)

  • Power: P=W/TP = W/TP=W/T -> [W] = [J .s^-1] = [kg .m^2 .s^-3] (Watt)

  • Electric charge: Q=I∗TQ = I*TQ=I∗T -> [C] = [A .s] (Coulomb)

  • Electric potential: F=EqF = EqF=Eq -> [V] = [N .C^-1] = [kg .m .s^-3 .A^-1] (voltage)

  • Magnetic flux density: ∇×E⃗=−∂B⃗∂t\nabla\times\vec{E} = -\frac{\partial\vec{B}}{\partial t}∇×​E​⃗​​=−​∂t​​∂​B​⃗​​​​ -> [T] = [E .m^-1 .s] = [kg .s^-2 .A^-1] (Tesla)

  • Magnetic flux: ψB=B⃗⋅S⃗\psi_B = \vec{B}\cdot\vec{S}ψ​B​​=​B​⃗​​⋅​S​⃗​​ -> [Wb] = [T .m^2] = [kg .m^2 .s^-2 .A^-1] (Weber)

  • Equivalent dose (Ionizing radiation)::: [Sv] = [J .kg^-1] = [m^2 .s^-2] (Sievert)
