Statistic Analysis

Author: DocNan Date: March 2, 2019  11:45:46 Category: Math

Statistic analysis is very important for judging the property of a large quantity of experiment data. For example, standard error can be used to calculate error bar.

Basic definition

Suppose we have lots of experiment data with discrete values as a finite series: x1,x2,x3,...,xNx_1, x_2, x_3, ..., x_Nx1​,x2​,x3​,...,xN​. Then the related statistic values are defined like these.

Mean value

μ=1N∑i=1Nxi\mu = \frac{1}{N}\sum_{i=1}^N x_i μ=N1​i=1∑N​xi​

Standard deviation (SD)

Standard error is a very important statistic value, because we usually choose error bar in this form during experimental data analysis.
